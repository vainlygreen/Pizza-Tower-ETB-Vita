# Pizza-Tower-ETB-Vita
a port of pizza tower early test build (2018) to the playstation vita

minimal issues such as crashing when choosing exit on the pause menu, and freezes when switching rooms super fast

best played with overclock but there is lag in the tutorial room, the main level runs fine afterwards but there are also palette issues present (specifically with the weenie enemies)

made with the etb decomp and gaymaker

pizza tower is by Tour De Pizza
